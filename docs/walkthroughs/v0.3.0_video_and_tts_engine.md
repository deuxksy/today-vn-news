# Walkthrough - Video & TTS Engine Implementation (v0.3.0)

FFmpeg 기반의 영상 합성 엔진과 고도화된 TTS 정제 로직을 구축한 단계입니다.

## 주요 성과 (Key Achievements)

### 1. 영상 합성 엔진 (`engine.py`)
- FFmpeg를 활용하여 TTS 오디오와 배경 영상을 합성하는 엔진 구현.
- **동적 길이 조정**: TTS 오디오 길이에 맞춰 영상 소스를 루프하거나 자르는 로직 추가.
- 하드웨어 가속(VideoToolbox/VAAPI) 초기 프레임워크 구축.

### 2. TTS 정제 및 최적화
- `edge-tts` 기반 한국어/베트남어 음성 생성 구조 확립.
- **텍스트 전처리**: 에모지, 특수문자 등 TTS 발음에 방해되는 요소를 제거하는 클리닝 로직 강화.

### 3. 데이터 구조화
- 확장 가능한 `SOURCES` 구조 도입.
- 마크다운 계층 구조(##, ###)를 통한 뉴스 리포트 가독성 확보.

## 기술적 결정 (Technical Decisions)

- **Single Source of Truth**: 모든 수집 및 로직 명세를 `ContextFile.md`에 집중시키고 코드는 이를 따르는 구조로 개편.
- **보안**: `.gitignore` 규칙을 세분화하여 프로젝트의 기밀성 강화.

## 검증 (Verification)
- 생성된 MP4 파일의 오디오-비디오 싱크 적합성 확인.
- TTS 음성의 명확성 및 자연스러움 검토.
