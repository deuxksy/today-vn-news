# Walkthrough - Deployment & Pipeline Activation (v0.4.0)

YouTube API 통합 및 실질적인 파이프라인 가동을 완료한 단계입니다.

## 주요 성과 (Key Achievements)

### 1. YouTube API 통합 (Step 4 완료)
- OAuth 2.0 기반 `uploader.py` 구현.
- `client_secrets.json`을 활용한 자동 영상 업로드 프로세스 정착.
- 보안 강화: API 키 및 자격 증명 파일의 Git 노출 원천 차단.

### 2. 뉴스 수집 범위 확대
- 수집 대상을 베트남 7대 일간지(`VnExpress`, `Tuổi Trẻ`, `Thanh Niên`, `Nhân Dân` 등) 전체로 확장.
- 환경/안전 실시간 정보 수집 엔진 완성.

### 3. 멀티 인프라 최적화 (v0.4.1)
- **Steam Deck 가속**: VAAPI 기기 경로(`-vaapi_device`) 명시적 지정으로 SteamOS 호환성 확보.
- **Mac Mini 가속**: VideoToolbox를 활용한 고해상도 렌더링 검증.

## 기술적 결정 (Technical Decisions)

- **SOP 동기화**: `ContextFile.md`에 정의된 큐레이션 우선순위(건강→IT→로컬)를 수집 코드 수준에서 강제로 일치시킴.
- **출력 정제**: 뉴스 리포트에서 AI가 생성하는 불필요한 인사말이나 메타 정보를 제거하는 Zero-Noise 로직 적용.

## 검증 (Verification)
- `main.py` 실행 시 뉴스 수집부터 유튜브 업로드까지 전 과정 자동화 성공 확인.
